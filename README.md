# 🚀 DroneOS

DroneOS is a real-time drone control system with mission planning, telemetry, and autonomous flight logic.

## 🔥 Features
- Real-time dashboard
- Mission planner (waypoints)
- Autopilot integration
- No-fly zone avoidance
- Telemetry system

## 🧪 Current Status
Working prototype (PX4 integration in progress)

## 🎯 Vision
A modular OS for autonomous drone control and fleet management
